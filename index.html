GameQuestBot

A chatbot for finding your next game.

About GameQuestBot

GameQuestBot is an interactive chatbot designed to provide video game recommendations based on provided information.

It utilizes the Dialogflow platform, which leverages natural language processing and machine learning to understand and respond to user queries.

GameQuestBot compares inputs with a database of video games populated with over 42000 games from the rawg.io API

Example Queries
Go ahead and tell GameQuestBot what you're looking for!

  • I want to try a brand new game
  • Some retro games please
  • I want to play a game like Minecraft
  • Looking for games on the PC or PS5
  • What are some good strategy games?
